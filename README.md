# RealTimeChat

Онлай чат на основе WebSockets с возможность так и анонимного общения, так и с помощью регистрации и аунтефиикации. Плюс минимальная поддержка обмена медиа-файлами 
Стек:
- backend (Fastapi + WebSockets)
- frontend (Vue + Socket.IO)

## Гайд по деплою на выделенный хост
Скопировать кодовую базу
git pull

Настроить переменные окружения

Добавить пользователя в группу docker (если не добавлена)
- sudo usermod -aG docker deploy

Применить изменения без перелогина
- newgrp docker

Проверить (выведет все контейнеры, без ошибок)
- docker ps

Запустить все контейнеры
docker compose up --build
