# Users Service 

## Гайд по интеграции в другие проекты

Для того чтобы интегрировать сервис для управления пользователя необходимо:
--- БАЗА ---
1) Настроить .env
2) Подстроить под docker compose проекта
3) Настроить маршрут для обращения к сервису. Например прописать в krakend api-gateway конфиг ендпоинты сервиса
4) Прописать интеграции для других сервисов

--- Данные ---
5) Задать роли пользователей в Users-service\application\core\schemas\roles.py
Например для приложения - чата, написать разрешение для управления чатом
(можно отправлять сообщения, искать по юзернейму всех пользователей / нельзя просматривать чужие переписки, отправялть от чужого имени сообщения)

+ При необходимости изменить пользовательские данные в модели БД:
Users-service\application\repositories\database\models\users.py
